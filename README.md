# `learn.el`

`learn.el` is a package to help run [learn-co](https://github.com/learn-co/learn-co) commands from Emacs.


<a id="orgd476860"></a>

## Commands

-   `learn-submit` runs `learn submit`
-   `learn-test` runs `learn test`
-   `learn-test-fail-fast` runs `learn test --fail-fast`
