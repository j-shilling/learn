* =learn.el=

=learn.el= is a package to help run [[https://github.com/learn-co/learn-co][learn-co]] commands from Emacs.

** Commands

*** ~learn-submit~ runs ~learn submit~

*** ~learn-test~ runs ~learn test~

*** ~learn-test-fail-fast~ runs ~learn test --fail-fast~
